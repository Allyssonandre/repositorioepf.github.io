Main Description

* Coleta dados das personas e suas características sociais.

* Uma amostra dos temas mais comentados e relevantes no momento, e um comparativo de qual plataforma os usuários são convergentes em um tema similar.

* Fazer uma métrica do engajamento nas plataformas escolhidas, com                         o auxílio das ferramentas Google Analytic e a Audience Insights(Facebook).

* Procura interagir com os participantes das plataformas, com o intuito de criar empatia. Buscando a coleta de insights necessário para a análise mais apurada do público alvo.
